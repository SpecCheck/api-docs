---
title: "Get Options"
description: "This endpoint fetches the options for a given lab."
---
```
POST https://api.speccheckrx.com/v1/options
```
<Note>The response changes based on what is inputted. `lab_name` and `account_number` are always required.</Note>

<AccordionGroup>
  <Accordion title="Getting Job Types, Materials, Services, and Suggestions">
    `material` and `lens_style` should not be set.
  </Accordion>

  <Accordion title="Getting Lens Styles for a Given Material">
    `material` is set, `lens_style` should not be set.
  </Accordion>

  <Accordion title="Getting All Add-Ons for a Given Material and Lens Style">
    `material` and `lens_style` should both be set to return ``
  </Accordion>
</AccordionGroup>
### Body

<ParamField body="lab_name" type="string" required="true">
  `Cherry Optical Lab`
</ParamField>

<ParamField body="account_number" type="string" required="true">
</ParamField>

<ParamField body="material" type="string">
 The relevant configuration options for a given material. If lens_style is also given, 
</ParamField>

<ParamField body="lens_style" type="string">
  Material must be set if lens_style is set.

### Response

<ResponseField name="job_types" type="list of strings">
  List of valid job types.
</ResponseField>

<ResponseField name="materials" type="list of strings">
  List of materials.
</ResponseField> 

<ResponseField name="suggested_values" type="dict">
  Suggested values are set by the lab and used for our frontend purposes as shortcuts.
</ResponseField> 

<ResponseField name="services" type="list of strings">
  List of available services.
</ResponseField> 

<ResponseField name="lens_styles" type="list of strings">
 List of valid lens styles.
</ResponseField>

<ResponseField name="lens_type" type="string">
 Lens type of the `lens_style` passed in.
</ResponseField>

<ResponseField name="tints" type="list of strings">
 Valid tint combinations. Returned if both material and lens_style are set.
</ResponseField>

<ResponseField name="addons" type="object">

 List of valid coatings. Returned if both material and lens_style are set.
