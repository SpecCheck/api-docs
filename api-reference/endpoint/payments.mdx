---
title: "Get Payments"
description: "This endpoint returns all payments processed through SpecCheck for a given lab."
---

```
POST https://api.speccheckrx.com/v1/payments
```

### Body

<ParamField body="lms" type="string" required="true">
  The name of the LMS.
</ParamField>

<ParamField body="lms_lab_id" type="string" required="true">
  The LMS lab ID.
</ParamField>

<ParamField body="start_date" type="string" required="true">
  The date to return payments from.
</ParamField>

<ParamField body="end_date" type="string" required="true">
  The date to return payments up to.
</ParamField>

### Response

<ResponseField name="charges" type="list of Charge Object">
  <Expandable title="Charge">
    <ResponseField name="account_number" type="string">
      The lab account number of the ECP who made this payment.
    </ResponseField>

    <ResponseField name="amount" type="float">
      The amount of the payment in major units.
    </ResponseField>

    <ResponseField name="created_at" type="string">
      ISO formatted date string of when the payment was made.
    </ResponseField>

    <ResponseField name="payment_method" type="string">
      The type of payment method used, example: `card`
    </ResponseField>

  </Expandable>
</ResponseField>


<RequestExample>

```bash Example Request
curl -G -H 'Authorization: bearer your_bearer_token' -H 'Client-Id: your_client_id' \
--data-urlencode start_date="2023-04-01T00:00:00+0000" --data-urlencode end_date="2023-04-17T23:59:59+0000" \
--data lms="DVIRx" --data lms_lab_id="1057148" 'https://api-staging.speccheckrx.com/v1/payments'
```

</RequestExample>
